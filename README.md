# School Database Lab

## Description

This project demonstrates how to design and implement a normalized PostgreSQL school database.

## Technologies

- PostgreSQL
- SQL
- Git
- GitHub

## Database Structure

- Students
- Teachers
- Courses
- Enrollments

## Features

- Relational database design
- Primary keys
- Foreign keys
- Data insertion
- JOIN queries
- Aggregate queries
- GROUP BY and HAVING
- Window functions
- Student ranking

## How to Run

Create a PostgreSQL database.

Execute:

```sql
\i schema/school_schema.sql
```

Load the data:

```sql
\i data/seed_data.sql
```

Run the queries inside the queries folder.
